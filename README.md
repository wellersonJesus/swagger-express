## Docker

```bash
code docker run -it --rm -p 5000:5000 -v $(pwd):/api -w "/
```

## Install

```bash
api" node bash
npm init -y
npm i express
touch app.js
```

## Dependences

```bash
sudo npm install -g nodemon
sudo npm i swagger-jsdoc swagger-ui-express
nodemon -L app.js => run api
```

## Host

```bash
#localHost
$ localhost://5000/api
```
